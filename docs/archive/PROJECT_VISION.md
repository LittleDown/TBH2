# Visão do Projeto

TBH2 é um Idle RPG compacto para permanecer aberto ao lado de outras aplicações.
O jogador acompanha um herói que luta, evolui e melhora seu equipamento de forma
automática.

## Princípios

- Progressão compreensível e constante.
- Interação mínima para manter o foco no trabalho principal do usuário.
- Interface pequena, legível e responsiva.
- Baixo consumo de recursos.
- Estado persistente entre sessões.

## Escopo da v0.1

A primeira versão entrega um ciclo completo de combate, experiência, level up,
loot, equipamento automático e save local.
