# Acessórios

## Objetivo

Definir os espaços de acessórios, suas famílias, atributos e papel dentro da progressão do TBH2.

Os acessórios representam a principal fonte de especialização do personagem.

Enquanto armas aumentam o poder ofensivo e armaduras aumentam a sobrevivência, os acessórios definem como a build funciona.

# Filosofia dos Acessórios

Os acessórios existem para criar escolhas.

Seu objetivo não é fornecer grandes quantidades de dano ou defesa.

Seu objetivo é modificar, amplificar e direcionar a build.

Os acessórios são a principal ferramenta de personalização do personagem.

Um acessório ideal deve fazer o jogador pensar:

> "Isso combina perfeitamente com minha build."

# Função dos Espaços

## Rings

Os anéis possuem foco em atributos específicos.

Exemplos:

* crítico;
* velocidade;
* dano elemental;
* regeneração;
* evasão;
* experiência.

Os anéis permitem ajustes finos da build.

---

## Amulet

O amuleto possui maior potencial de impacto.

Ele pode fornecer:

* atributos elevados;
* efeitos lendários;
* modificadores exclusivos;
* bônus de classe.

Os melhores amuletos devem ser itens extremamente desejados.

# Progressão de Poder

Os acessórios possuem menor poder bruto que armas e armaduras.

Porém possuem maior capacidade de especialização.

Exemplo:

Arma:

+120 dano

Anel:

+20% chance crítica

Embora o número seja menor, o impacto na build pode ser maior.

---

# Efeitos Especiais

Os acessórios são candidatos naturais para efeitos únicos.

Exemplos:

* +10% XP obtida;
* +15% ouro encontrado;
* +5% chance de item raro;
* Recupera vida ao derrotar inimigos;
* Reduz tempo de recarga de habilidades.

Esses efeitos devem ser raros e desejáveis.

---

# Poder

Poder representa a qualidade geral do acessório.

Ele considera:

* atributos;
* raridade;
* afixos;
* efeitos especiais.

Serve apenas como comparação rápida.

---

# Build Score

Build Score representa o valor real do acessório para a build atual.

Ele considera:

* atributos relevantes;
* sinergias;
* classe;
* habilidades;
* tags.

Build Score possui prioridade sobre Poder.

# Filosofia de Progressão

No início do jogo os acessórios possuem impacto moderado.

Conforme a build evolui, eles se tornam cada vez mais importantes.

No Endgame:

* armas definem o dano base;
* armaduras definem sobrevivência;
* acessórios definem eficiência.

Os melhores acessórios devem ser capazes de transformar completamente uma build.

---

# Regras

* Todo acessório deve possuir uma função clara.
* Não criar acessórios que apenas repetem atributos de armas.
* Rings e Amulets devem possuir identidades diferentes.
* Build Score possui prioridade sobre Poder.
* Efeitos especiais devem ser raros e controlados.
* Acessórios não devem substituir habilidades ou classes.
