# Sprites temporários

Assets raster temporários para a Vertical Slice visual do TBH2.

Foram gerados com a ferramenta integrada de geração de imagens e processados
localmente para remover o fundo chroma key, recortar cada célula e normalizar os
sprites em telas transparentes de 128x128.

Conteúdo:

- `hero.png`
- `goblin.png`
- `wolf.png`
- `skeleton.png`
- `boss.png`
- `loot_chest.png`

Prompt-base:

> Seis sprites de fantasia em pixel art 16-bit, dispostos em uma grade 3x2:
> aventureiro, goblin, lobo, esqueleto, capitão morto-vivo e baú de tesouro;
> silhuetas legíveis, escala consistente, sem texto, em fundo chroma key verde.

Esses arquivos são placeholders e devem ser substituídos por arte definitiva
antes de uma versão comercial.
