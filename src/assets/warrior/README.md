# Frames do guerreiro

Frames fornecidos no projeto JavaScript de referência `thb2-main.zip` e usados
somente na cena animada do TBH2 em Python.

O controlador visual genérico utiliza:

- `walk1.png` e `walk2.png` durante exploração e aproximação;
- `idle.png` como repouso e fallback de `combat_idle`;
- `attack1.png` e `attack2.png` no golpe do herói;
- `hit.png` ao receber dano;
- `victory.png` durante recompensa;
- `dead.png` como alias de `defeat`.

Os arquivos `block.png` e `front.png` permanecem disponíveis para evolução
visual futura. Nenhuma regra ou sistema do projeto JavaScript foi convertido.
